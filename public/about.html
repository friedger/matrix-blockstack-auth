OI Chat

OI Chat is a matrix service dedicated to Blockstack users. Your Blockstack account is your matrix account. The first 1000 are given away for free!

Features (Chat App)

All the goodness from the default Riot app like 1:1 and group chats, end-to-end encryption, video calls, .. plus

  • Login with Blockstack
  • Export/import of e2e keys on Gaia (coming soon)
  • Device verification via Gaia (coming soon)

Features (Server)

All the goodness from the default matrix server like decentralization, openness, application integrations, ... plus

  • No passwords stored on server
  • Send messages from supported Blockstack apps
  • Manage app permissions (coming soon)
  • Support upcoming standardizations (like MSC1768)

You can use OI Chat with any matrix client you like. We provide a web client (OI Chat Web) that leverage features of your blockstack account and simplifies login

Login with Other Clients

If you choose a different client than OI Chat Web you need to use a one-time login.

  • Sign in with Blockstack here
  • Click "Generate One-Time Login"
  • Copy and paste
    • username (your blockstack address) and
    • password (details to authenticate you using Gaia storage)
    into the login screen of your client
