OI Chat is a dedicated Matrix home server for Blockstack users. Your blockstack account is your matrix account!

Features (Home Server)

  • No passwords stored on server
  • Blockstack apps can post in your name
  • Manage app permissions (coming soon)

Features (Client)

  • Login with Blockstack
  • Export/import of e2e keys on Gaia (coming soon)
  • Device verification via Gaia (coming soon)

You can use OI Chat with your preferred matrix client. However, we provide a web client (OI Chat Web) that leverage features of your blockstack account.

If you use a different client than OI Chat Web you can use a one-time login.
(Your user name is your blockstack identity address together with a "password" that helps the home server to authenticate you - using your Gaia storage)

  • Sign in with Blockstack
  • Click "Generate One-Time Login"
  • Copy and paste username and password into the login screen of your client
